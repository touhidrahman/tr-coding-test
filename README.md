# TrCodingTest

Built using [Angular CLI](https://github.com/angular/angular-cli) version 9.x.

Tech staacks used:

-   Angular v9
-   Tailwindcss v1.2.0

## Run

-   `git clone https://github.com/touhidrahman/tr-coding-test`
-   `cd tr-coding-test`
-   `npm i` (First time only)
-   `npm start`
-   Navigate to `http://localhost:4200/`.
